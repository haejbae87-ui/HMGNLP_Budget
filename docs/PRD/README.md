## PRD 작성 가이드

신규 기능 개발 시 `prd_manager` 스킬 사용:

```
# 신규 개발 시 (사전 설계)
→ prd_manager 스킬: 요구사항 수집 → PRD 초안 작성 → 코드 개발

# 기존 코드 역추적 시
→ prd_engineer 스킬: 코드 스캔 → 비즈니스 로직 추출 → PRD 생성
```
