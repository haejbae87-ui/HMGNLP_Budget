# 예산 배분 통합 드릴다운 — 요구사항 정의서 (PRD)

> **도메인**: 예산관리 (Budget Distribution Drilldown)
> **관련 파일**: `bo_allocation.js`, `bo_styles.css`
> **최초 작성**: 2026-05-01
> **최종 갱신**: 2026-05-01
> **상태**: 🟡 구현 중

---

## 1. 기능 개요

예산 배분 화면을 "큰 통장 → 작은 통장 이체" 메타포의 **통합 드릴다운 UI**로 전면 재설계.
기존 5탭(현황/기초배정/팀배분/이관/이력)을 3탭(현황/최초할당/예산배분/이력)으로 간소화하고,
계정→교육조직→팀→(개인) 4레벨 계층을 하나의 드릴다운 화면에서 관리.

## 2. 사용자 스토리

> "총괄담당자는 예산 계정의 마스터 통장에서 교육조직·팀 통장으로 예산을 배분할 수 있다."
> "운영담당자는 관할 교육조직 통장에서 하위 팀 통장으로 예산을 배분할 수 있다."
> "배분 확정 후 수정이 필요하면 회수 기능으로 역이관할 수 있다."
> "모든 배분/회수 이력은 Audit Trail로 자동 기록된다."

## 5. 예외 처리 및 엣지 케이스

| 케이스 | 처리 방식 |
|---|---|
| 배분 합계 > 마스터 잔액 | 실시간 빨간색 경고 + 제출 비활성화 |
| 회수 시 (약정+집행) 미만으로 남음 | 최소값 제한, 회수 불가 안내 |
| 교육조직 0개 (VOrg 미설정) | 빈 상태 안내 + 설정 유도 |
